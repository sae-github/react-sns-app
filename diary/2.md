# エラー・悩んだ点

- フロント側から Express で作成したエンドポイントへアクセスすると CORS 対応していないとエラーがでた

(解決策)
cors をインストールし、server.js へ以下を記載。

全ての API を許可する様にした

```js
const cors = require("cors");
app.use(cors());
```
